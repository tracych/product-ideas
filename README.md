# product-ideas

Side-project sandbox &mdash; small, self-contained product experiments.

## Projects

### [maze-trickster/](./maze-trickster)

A 10-level jumping-maze browser game where the maze lies. Floors vanish, doors deceive, gravity flips, and inputs mirror. Single-file HTML, no dependencies &mdash; open [`maze-trickster/index.html`](./maze-trickster/index.html) in any browser to play.

### [jumping-ball-runner/](jumping-ball-runner)
 
A 3D endless-runner browser game where a bouncing ball dodges and leaps over oncoming obstacles, with score tracking and a persistent best score. Single-file HTML built with Three.js (loaded from CDN) — open `jumping-ball-runner/index.html` in any browser to play.
